📦 הוראות העלאה ל-Railway:

1. עבור לכתובת https://railway.app
2. הירשם או התחבר עם חשבון GitHub או Gmail
3. לחץ על "New Project" > "Deploy from GitHub" או "Deploy from Template"
4. בחר "Deploy from GitHub" והעלה את הקבצים מהמחשב שלך (main.py, requirements.txt)
   או גרור את התיקיה לתוך עורך Railway.
5. לחץ על Settings > Variables והוסף משתנה:
   שם: BOT_TOKEN
   ערך: (הטוקן שקיבלת מה-BotFather)

6. חזור ל-Deploy ולחץ "Deploy Now"
7. אחרי כמה שניות תראה שהבוט שלך רץ 🎉

כדי לבדוק שהוא עובד, תכתוב לו /start או תכניס אותו לקבוצה.
